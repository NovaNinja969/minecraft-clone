# Minecraft Clone

A browser-based Minecraft clone built with Three.js and TypeScript.

## Features

- ✅ Voxel-based world rendering
- ✅ Block placement and breaking
- ✅ Procedural terrain generation with Perlin noise
- ✅ Multiple biomes
- ✅ Player movement and camera controls
- ✅ Inventory system
- ✅ Crafting system
- ✅ Entity/Mob system
- ✅ Dynamic lighting
- ✅ Chunk loading/unloading

## Installation

```bash
git clone https://github.com/NovaNinja969/minecraft-clone.git
cd minecraft-clone
npm install
```

## Development

```bash
npm run dev
```

Then open `http://localhost:8080` in your browser.

## Build

```bash
npm run build
```

## Controls

- **WASD** - Move
- **Space** - Jump
- **Shift** - Sprint
- **Left Click** - Break block
- **Right Click** - Place block
- **E** - Open inventory
- **ESC** - Pause menu
- **Mouse** - Look around
